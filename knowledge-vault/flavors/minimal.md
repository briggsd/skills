# Flavor: Minimal

Bare skeleton. Start here if you want to shape the vault yourself as real content arrives.

## Default folders

- **Daily/** — Daily notes.
- **Projects/** — Active projects.
- **Notes/** — Loose thoughts, ideas, drafts.
- **Resources/** — Reference material. Includes `archive/`.
- **_templates/** — Note templates.

## Suggested maintenance rules

R1, R2, R3, R4, R5, R7, R8. Skip R6 (archiving) until you have things to archive.

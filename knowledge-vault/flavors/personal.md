# Flavor: Personal

For individuals managing their life, family, hobbies, and personal projects.

## Default folders

- **Daily/** — Daily notes, one file per day (YYYY-MM-DD.md). Session logs live here.
- **People/** — Family, friends, professional contacts. Grouped by relationship.
- **Projects/** — Active personal projects (learning, side builds, life initiatives).
- **Notes/** — Loose thoughts, ideas, drafts that aren't tied to a project yet.
- **Goals/** — Life goals, yearly/quarterly intentions, habit tracking.
- **Health/** — Fitness, diet, sleep, medical notes (non-sensitive).
- **Finances/** — Personal budget, saving goals, expense patterns.
- **Hobbies/** — Recipes, reading lists, music, craft projects, etc.
- **Travel/** — Trip planning, packing lists, places visited.
- **Resources/** — Saved articles, PDFs, reference material. Includes `archive/`.
- **_templates/** — Note templates.

## PII rules override (People/)

Relaxed compared to work flavor. OK to store:
- Names, nicknames, relationship
- Phone, email, address (home is fine for close family/friends)
- Birthdays, anniversaries
- Favorite things, gift ideas, preferences
- Medical notes if the user wants (e.g., allergies, dietary restrictions)

NOT OK to store:
- Passwords, account numbers, SSN, government IDs
- Anything covered by the root `## What NOT to put here` rule

## People groups (examples)

- `Family/` — immediate and extended family
- `Friends/` — close friends
- `Professional/` — colleagues, mentors, service providers (doctor, accountant, etc.)
- `Neighbors/`
- `Community/` — clubs, volunteer groups

## Suggested maintenance rules

All R1-R8. Skip R9 unless using the productivity plugin.

# Flavor: Work

For professionals managing business context, teams, and projects at work.

## Default folders

- **Context/** — Business strategy, ICP, brand, positioning, organization.
- **Daily/** — Daily notes (YYYY-MM-DD.md). Session logs live here.
- **Departments/** — Functional areas (sales, marketing, engineering, etc.) — spin up on demand.
- **Intelligence/** — Competitive research, industry notes, market signals.
- **Onboarding/** — Materials for bringing new team members up to speed.
- **Projects/** — Active work projects. Each has CLAUDE.md + project_current_state.md.
- **Resources/** — Reference material, saved docs, articles. Includes `archive/`.
- **Tasks/** — Task tracking (or defers to productivity plugin if installed).
- **Team/** — Colleagues, partners, contractors. Grouped.
- **Team Schedules/** — Availability, working hours, time zones.
- **_templates/** — Note templates.

## PII rules override (Team/)

Strict. OK to store:
- Name, role, company, work email
- Time zone, working hours, LinkedIn URL
- Working-style notes, meeting history, project involvement

NOT OK to store:
- Personal phone, home address, emergency contacts
- Health info, family details, financial info
- Anything the person wouldn't share in a professional context

## Team groups (examples)

- `Internal/` — direct team, coworkers
- `External/` — contractors, vendors, partners
- `Clients/` — client contacts
- `Advisors/` — advisors, mentors, board

## Suggested maintenance rules

All R1-R9. R9 (productivity plugin deferral) recommended since many work users install it.

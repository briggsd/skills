# _templates/

Starter files for new notes. Copy, don't edit-in-place.

- `daily-note.md` — daily log entry
- `project-readme.md` — a project's root CLAUDE.md
- `project-current-state.md` — rolling project state file
- `person.md` — person/contact file
- `meeting-note.md` — meeting capture
- `decision-log.md` — standalone decision record
- `weekly-review.md` — end-of-week reflection
